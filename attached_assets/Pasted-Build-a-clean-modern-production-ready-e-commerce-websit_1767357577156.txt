Build a clean, modern, production-ready e-commerce website for the brand "Choco Blossom India" using HTML, CSS and JavaScript.

The store sells handmade chocolates, gift hampers, personalized gift combos and festive gift boxes. The design should look elegant, premium, gifting-oriented and trustworthy.

IMPORTANT CODE REQUIREMENTS:
• Use semantic HTML5 structure
• Use external CSS (no inline styles)
• Create reusable CSS classes
• Make it fully responsive (mobile-first)
• Use clean typography, spacing and section gaps
• Keep code readable and organized
• Include hover states for buttons and cards
